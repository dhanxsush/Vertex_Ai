# Vertex_Ai
# Semantic Vector Search using AlloyDB + Vertex AI

This repository contains the output of a semantic vector search experiment
performed using AlloyDB integrated with Vertex AI text embeddings.

The search retrieves records based on semantic similarity rather than
traditional keyword matching.

## Technical overview
- Database: AlloyDB (PostgreSQL-compatible)
- Embeddings: Vertex AI Gemini text embeddings (dense vectors)
- Search method: Vector similarity search (semantic intent)

This repository shares only the query output for demonstration purposes.
No cloud resources, credentials, or proprietary configurations are included.
